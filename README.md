# YourCarYourWay Chat proof of concept!


# Installation

clone the project in your IDE.

 git clone git@github.com:peasch/ProofOfChat.git

## DataBase :

$make sure to configure a database, create a model, an run the dump files.

 don't forget to set the DB credentials :
- in the run configuration
- environment variables
- DB_USERNAME / DB_PASSWORD

## build & run the back end
cd /back/

 mvn clean install

run BackendApplication

## build and run the front end

cd /front/

npm start

navigate to http://localhost:4200/. 

# CHAT !

open several browsers to http://localhost:4200/, and speak as you want!
